factories that produce production-grade software with coding factory workers.

## Alternatives Considered

### Amadeus connects to Hooin vs Hooin connects to Amadeus

Although having Hooin directly wake Amadeus looks cleaner, Amadeus is a highly
stateful service with expensive startup. A given Amadeus worktree must only be
started by a single instance globally, which makes it hard for Hooin to discover
an already-running Amadeus that's connected to a different Hooin instance.
